Attributes and other stuff

Attributes are how you set properties of an element. They are inserted in the first element of the tag pair, after a space. You put the atribute_name="atribute value"

The mean attributes that almost each class have are the following: Id,Title,Class and Style.

The Id is very important becouse allows you to uniquely identify an element inside the HTML. Said that, you should not name two tags with the same Id. The Title shows in a box whenever the user place the cursor of the element in the page. The Class has to do with what we call cascade style sheet or CSS, we will leave this one for another moment.

Do you know that in some languages we read from the right to the left? This is the case in Hebrew, for example. HTML can set the direction of the text usin what we call Internationalization attributes such as dir. The default value of dir is rtl, however if you are in writing in hebrew you can set it to ltr. By applying this attribute value to the html tag, it will be set to the entire document.
